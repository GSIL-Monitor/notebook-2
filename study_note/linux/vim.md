## 3. 块操作
- 块编辑插入
```shell
# 块编辑插入
ctrl+v
选择块
选完后按大写的I，进入编辑模式
编辑完后按esc 完成块的编辑
```
- 块删除
```shell
# 块删除
chtr + v
选这块 - 要删除的
按delete 完成删除
然后保存
```
